Häufig gestellte Fragen zu ReGENESIS

Was ist ReGENESIS?

ReGENESIS ist eine ständig aktualisierte Fassung der Statistiken von Bund und Ländern. Die Tabellen werden dabei in Formaten angeboten, die für eine maschinelle Auswertung besonders gut geeignet sind.

Statistikdaten von Bund und Ländern liegen zumeist als aggregierte Berichte vor, die deren komplexer Tabellenaufbau kaum für eine automatische Auswertung oder die Nutzung in interaktiven Grafiken zugänglich sind.

ReGENESIS bezieht seine Daten daher aus dem Massenexport verschiedener Instanzen des GENESIS-Systems, das vom Statistischen Bundesamt zur Verwaltung statistischer Daten entwickelt wird. So werden statistische Daten im Rohformat, also sog. Datenquader, bereitgestellt.

Ist dies ein offizielles Angebot?

Nein. Die Daten entstammen aber einer amtlichen Quelle und sind im Inhalt nicht verändert.

Sind die Daten frei verfügbar?

Der §5 des Urheberrechts schliesst amtliche Statistiken vom Urheberrecht aus und stellt damit sicher, dass diese frei genutzt werden können. Die abgeleiteten Tabellen auf ReGENESIS werden ebenfalls ohne Einschränkungen unter den Bedingungen einer CC0-Lizenz weitergegeben.

Falls Sie dennoch auf ReGENESIS als Quelle verweisen wollen, dann können Sie das zum Beispiel mit folgendem HTML-Snippet tun:

<a href="http://regionalstatistik.de" target="_blank">Regionalstatistik.de</a> via <a href="http://regenesis.pudo.org/" target="_blank">ReGENESIS</a>.

Was sind Statistiken, Tabellen und Attribute?

Zur Speicherung der Daten verwenden die Ämter ein komplexes Datenmodell, welches sich für fortgeschrittene Analysen eignet. Im Sinne der Nutzbarkeit versucht ReGENESIS dieses zu vereinfachen und einige der verwendeten Fachbegriffe durch gängigere Wörter auszuwechseln:

Statistiken sind Erhebungen, deren Erfassung gesetzlich geregelt ist. Dabei werden verschiedene Merkmale erhoben, allerdings nicht vollständig veröffentlicht um die Anonymisierung der Daten nicht zu gefährden.

Tabellen (auch Datenquader) sind Auszüge aus einzelnen Statistiken, die als Download angeboten werden. Sie enthalten zumeist nur einen Teil der Daten, die insgesamt für die Statistik erhoben wurden. Zudem werden Tabellen auf verschiedenen geographischen Niveaus angeboten, also z.B. nach Bundesland, Kreis oder Gemeinde.

Attribute (auch Merkmale, Messwerte, Dimensionen) sind mit den Spalten einer Tabelle vergleichbar. Sie beschreiben z.B. wann, wo und in welcher Untergruppe welche Werte beobachtet wurden.

Wie oft werden die Daten aktualisiert?

Im Regelbetrieb werden wie Daten wöchtenlich aus den ursprünglichen Datenquellen aktualisieren.

Gibt es eine Programmierschnittstelle (API)?

Noch nicht, aber eine einfach zu bedienende API auf der Basis von JSON wird hoffentlich eine der nächsten Funktionen sein, die wir anbieten. Damit können Statistiken aus ReGENESIS direkt in Visualisierungen eingebunden werden, die somit immer aktuell bleiben.
